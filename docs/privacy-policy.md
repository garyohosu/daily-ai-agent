# プライバシーポリシー

日刊AIエージェント（以下「当サイト」）では、サービス改善および広告配信のため、利用情報を取得する場合があります。

## 取得しうる情報

- 閲覧ページ、アクセス日時、リファラー
- 端末/ブラウザ等の技術情報
- 広告配信・解析サービスにより取得される情報

## 利用目的

- コンテンツ品質の改善
- 障害調査、不正利用対策
- 広告配信および効果測定

## 広告・解析について

当サイトでは Google AdSense 等の外部サービスを利用する場合があります。これらのサービスは Cookie などを利用して情報を扱うことがあります。

## 外部サービス

GitHub Pages など、外部サービスの基盤を利用しています。各サービスでの取り扱いは提供者の規約に従います。

## 免責事項

当サイトの情報は正確性に配慮しますが、完全性・有用性を保証するものではありません。
